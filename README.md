# Autobazar

Jednoduchá webová aplikace „autobazar“ vytvořená v čistém HTML, CSS a JavaScriptu.  
Cílem projektu je ukázat práci s třídami v JavaScriptu a základní principy OOP.

Aplikace umožňuje přidávat auta pomocí formuláře, zobrazit jejich seznam a jednotlivá auta ze seznamu mazat. Součástí je také výpočet počtu aut a průměrné ceny.

Projekt je rozdělen do více souborů – zvlášť třída pro auto, třída pro správu autobazaru a hlavní skript aplikace.

### Spuštění
Otevřete soubor `index.html` v internetovém prohlížeči.
